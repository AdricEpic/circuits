Module bridge

Bridge

Bridge Component to Bridge one or more components in a single System. That is, events in System A bridged to System B are shared. For example:

A <--> Bridge <--> B

Events that propagate in A, will propagate to B across the Bridge. Events that propagate in B, will propagate to A across the Bridge.

When the Bridge is created, it will automatically attempt to send a Helo Event to any configured nodes or on a broadcast address if no nodes are initially configured. The default Bridge implementation uses the UDP protocol and as such events cannot be guaranteed of their order or delivery.
